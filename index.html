Remind myself of how to complete repository


  1. On GitHub click create repository by clicking new on the dashboard.
  2. Give the repository a name.
  3. Complete the description section.
  4. Click on create repository.
  5. Create an index.html file that contains the html
  6. Name the file with the .html.
  7. Enable GitHub pages via settings and scroll to Pages.
  8. Navigate to your website.
